Multi-Start Levenberg-Marquardt Optimization

האלגוריתם מנסה למצוא את הפרמטרים המסלוליים (a, e, i, Ω, ω, M₀) שממזערים את ה-RMS residual בין המיקומים החזויים לתצפיות פיאצי.

Forward Model: params → M(t) → Kepler eq. → E → orbital pos → ecliptic → equatorial → RA/Dec

Loss Function: soft_l1 robust loss on residuals in arcseconds, with cos(δ) correction on ΔRA.

Multi-Start: 11 initial guesses (1 manual + 10 random) to escape local minima. Only one start (Random #8) finds the global minimum with RMS ≈ 3.4".

Gradient Display: ∂L/∂θ computed via finite differences, showing the descent direction in parameter space.

Based on 19 observations by Giuseppe Piazzi, Jan-Feb 1801.
